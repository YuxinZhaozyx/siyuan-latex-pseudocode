# siyuan-latex-pseudocode

一款让你在思源笔记中优雅地书写和展示伪代码的挂件。

## 功能特征

- [x] 无网络离线使用
- [x] 设置伪代码块的显示风格
- [x] VSCode风格的编辑器
- [x] 语法高亮
- [x] 代码自动补全
- [x] 错误提示
